# 1. El entorno de programación python

## El entorno de programación python

Python es un lenguaje de propósito general, fácil de leer y rápido de aprender. Se ejecuta con un intérprete, por lo que permite probar cambios de forma inmediata y depurar con rapidez.

Es multiparadigma: puedes programar con enfoque imperativo, orientado a objetos o funcional según el problema. Esta flexibilidad lo hace útil en proyectos pequeños y grandes.

En desarrollo web, Python destaca por frameworks como Django y Flask. Django incluye muchas herramientas listas para usar (autenticación, ORM, panel administrativo y seguridad), lo que acelera la construcción de aplicaciones.

También es multiplataforma: el mismo código suele funcionar en Windows, macOS y Linux con pocos cambios. Además, su ecosistema de paquetes en PyPI cubre necesidades comunes como web, datos, automatización y pruebas.

## Instalación y configuración de python

Para programar en Python, primero debes instalar el intérprete. Descarga la versión estable desde el sitio oficial de Python para tu sistema operativo.

En Windows, recuerda marcar la opción "Add Python to PATH" durante la instalación. En macOS y Linux, puedes usar gestores de paquetes como brew o apt si necesitas una versión específica.

Verifica la instalación con `python --version` o `python3 --version`. Luego confirma que `pip` esté disponible con `pip --version` y actualízalo con `pip install --upgrade pip`.

Como buena práctica, trabaja con entornos virtuales (`python -m venv`) para aislar dependencias por proyecto y evitar conflictos entre versiones.

## La consola de comandos y el entorno interactivo

El intérprete interactivo de Python permite ejecutar instrucciones al instante. Es ideal para aprender, hacer pruebas rápidas y depurar fragmentos pequeños.

Se abre desde terminal con `python` o `python3`. Para salir puedes usar `exit()` o los atajos del sistema (Ctrl + D en Unix, Ctrl + Z en Windows).

Herramientas como `help()`, `dir()` y `type()` te ayudan a explorar objetos y módulos sin salir de la consola. Para una experiencia más completa, IPython y Jupyter ofrecen autocompletado, mejor visualización y notebooks.

## Archivos .py y .pyw: escritura y ejecución de scripts en python

Los programas en Python se guardan normalmente en archivos `.py`. Estos archivos se pueden ejecutar desde terminal o importar en otros módulos para reutilizar código.

Para ejecutarlos, usa `python nombre_archivo.py` o `python3 nombre_archivo.py`, según tu sistema. Esto corre el script completo en orden secuencial.

En Windows también existe `.pyw`, pensado para aplicaciones gráficas que no necesitan mostrar consola.

Organizar tu proyecto en varios archivos `.py` mejora la mantenibilidad y facilita pruebas, reutilización y escalabilidad.

## Sincronía de la ejecución en python

Por defecto, Python ejecuta instrucciones de forma secuencial: una línea después de otra. Este comportamiento facilita seguir el flujo y encontrar errores.

Cuando necesitas concurrencia, puedes usar `threading`, `multiprocessing` o `asyncio`.

`threading` es útil para tareas de entrada/salida, pero el GIL limita la ejecución paralela real de código Python en CPU. Para trabajo intensivo de CPU, `multiprocessing` suele ser mejor porque usa procesos separados.

`asyncio` permite manejar muchas operaciones I/O sin bloquear el programa (por ejemplo, múltiples conexiones de red), mejorando tiempos de respuesta.

## El administrador de paquetes PIP

`pip` es el gestor de paquetes de Python. Permite instalar, actualizar y desinstalar bibliotecas publicadas en PyPI.

Su uso evita reinventar soluciones y acelera el desarrollo. En proyectos colaborativos, es clave definir dependencias en `requirements.txt` para que todos trabajen con versiones compatibles.

Recomendación práctica: crea un entorno virtual por proyecto y ejecuta `pip` dentro de ese entorno. Así reduces errores por dependencias globales y mejoras la reproducibilidad al desplegar.

## Conceptos básicos de python

Python se caracteriza por una sintaxis clara y por el uso obligatorio de indentación para definir bloques de código.

Con `print()` puedes mostrar resultados durante pruebas o depuración. También usarás operaciones aritméticas básicas en casi cualquier programa.

Las variables no requieren declarar tipo de forma explícita: Python usa tipado dinámico y determina el tipo en ejecución.

Los comentarios con `#` ayudan a documentar decisiones importantes y facilitan el mantenimiento del código.

## Sintaxis básica en python

La sintaxis de Python prioriza legibilidad. Los bloques se delimitan con indentación, no con llaves.

El lenguaje soporta tipos como `int`, `float`, `str` y `bool`, y usa tipado dinámico. Por eso, el tipo puede cambiar según el valor asignado.

Para nombrar funciones y variables se recomienda `snake_case`. Esta convención mejora la lectura y sigue el estilo oficial de Python (PEP 8).

Estructuras como `if`, `for` y `while` permiten controlar el flujo con una sintaxis simple y consistente.

## Nombres de variables en python

Los nombres de variables deben empezar con letra o `_`. Pueden contener letras, números y guiones bajos, pero no espacios ni caracteres especiales.

Python distingue mayúsculas y minúsculas (`edad` y `Edad` son variables distintas). Usa nombres descriptivos en `snake_case` para que el código sea más claro.

No uses palabras reservadas como nombres (`if`, `for`, `return`, etc.), porque generan errores de sintaxis.

Una nomenclatura coherente mejora la lectura, el mantenimiento y el trabajo en equipo.

## Métodos en python

Un método es una función asociada a un objeto o clase. Permite agrupar comportamiento junto con los datos que ese comportamiento manipula.

Los métodos de instancia usan `self` y operan sobre un objeto concreto. Los métodos de clase usan `@classmethod` y reciben `cls` para trabajar sobre la clase. Los métodos estáticos (`@staticmethod`) no dependen ni de instancia ni de clase.

Elegir bien el tipo de método mejora la organización del código y evita responsabilidades mezcladas.

## Identación en python

La indentación define la estructura del programa en Python. Si se indenta mal, el código no ejecuta.

La recomendación es usar 4 espacios por nivel y mantener consistencia en todo el archivo. Mezclar tabs y espacios puede provocar errores difíciles de detectar.

Más que un estilo visual, la indentación en Python es parte de la sintaxis.

## Comentarios en python

Los comentarios son texto que Python ignora al ejecutar. Se usan para explicar decisiones, supuestos o partes complejas.

El comentario de línea usa `#`. Las comillas triples suelen usarse como docstrings para documentar funciones, clases o módulos.

La mejor práctica es combinar código claro con comentarios puntuales: ni ausencia total ni exceso de explicaciones obvias.

## Manejo de módulos en python

Un módulo es un archivo Python reutilizable que puede contener variables, funciones y clases.

Se importa con `import modulo` o `from modulo import elemento`. La primera forma mejora legibilidad y evita choques de nombres; la segunda es útil cuando necesitas pocos elementos.

Python incluye módulos estándar como `math`, `datetime` y `os`. También puedes crear módulos propios para separar responsabilidades y reducir duplicación.

## Módulos como scripts en python

Un archivo `.py` puede servir como módulo importable y también como script ejecutable.

Para separar ambos usos, se emplea `if __name__ == "__main__":`. El código dentro de ese bloque solo corre cuando ejecutas el archivo directamente.

Este patrón permite probar funciones sin romper su reutilización. Si necesitas argumentos por terminal, puedes apoyarte en módulos como `sys` o `argparse`.

## Ruta de búsqueda de módulos en python

Cuando haces un `import`, Python busca módulos en este orden general: carpeta actual del proyecto, rutas de `PYTHONPATH`, biblioteca estándar y paquetes instalados.

La lista real de búsqueda está en `sys.path`. Entenderla ayuda a resolver errores como `ModuleNotFoundError`.

Buena práctica: evita modificar `sys.path` de forma manual salvo casos puntuales. Es preferible organizar bien el proyecto y usar entornos virtuales para controlar dependencias.

## Archivos compilados en python

Al importar un módulo, Python genera bytecode (`.pyc`) y lo guarda en `__pycache__`. Esto acelera importaciones futuras.

Si el archivo fuente cambia, Python recompila automáticamente ese bytecode.

En la práctica diaria no necesitas gestionar estos archivos manualmente, pero conviene saber qué son para entender mejor el comportamiento del intérprete.

## La función DIR() en python

`dir()` muestra nombres de atributos y métodos disponibles en un objeto, módulo o clase.

Sin argumentos, lista identificadores del entorno actual. Con un objeto, ayuda a explorar qué puede hacer ese objeto.

Es una herramienta útil para aprendizaje y depuración rápida en consola interactiva.

## Manejo de paquetes en python

Un paquete es una carpeta que agrupa módulos relacionados para organizar mejor el código.

En muchos proyectos, `__init__.py` se usa para marcar y configurar el paquete. Puede estar vacío o exponer elementos importantes.

Los paquetes facilitan escalabilidad, reutilización y distribución de librerías instalables con `pip`.

## Importación y referencias internas en paquetes

Dentro de un paquete puedes usar importaciones relativas (`from . import modulo`) o absolutas (`import paquete.modulo`).

La importación absoluta suele ser más explícita en proyectos grandes. La relativa es útil para mantener cohesión interna del paquete.

Usar bien estas referencias reduce conflictos de nombres y evita errores de importación cuando el proyecto crece.

## Paquetes en múltiples directorios

En proyectos grandes, un paquete puede dividirse en varios subdirectorios para separar dominios y responsabilidades.

Mantener una estructura clara de carpetas y nombres evita importaciones frágiles. Según el caso, `__init__.py` puede ayudar a controlar la exposición de módulos.

Si la organización del proyecto es correcta y se ejecuta desde el entorno adecuado, las importaciones serán más estables y fáciles de mantener.

# 2. Tipos de datos y sentencias básicas

## Manejo de Python

### Tipos de datos y sentencias básicas

1. Números

Python maneja varios tipos numéricos, incluyendo:
- Enteros (int): Representan números sin decimales.
- Flotantes (float): Permiten manejar valores con decimales.
- Números complejos (complex): Contienen una parte real e imaginaria.

Los números en Python pueden utilizarse para realizar operaciones matemáticas básicas y avanzadas. También es posible trabajar con módulos como `math` y `random` para realizar cálculos más complejos y generar valores aleatorios.

Además, Python admite la conversión entre tipos numéricos. Por ejemplo, un número entero puede convertirse en flotante o viceversa utilizando `float()` y `int()` respectivamente. Esta flexibilidad es útil en cálculos precisos.

Las operaciones matemáticas incluyen suma (+), resta (-), multiplicación (*), división (/), división entera (//), módulo (%) y exponenciación (**). Estas permiten manipular los valores numéricos eficientemente.

El uso de operadores matemáticos en Python también incluye el uso de funciones avanzadas con la librería math, que proporciona métodos para realizar cálculos trigonométricos, logarítmicos y exponenciales.

Python maneja la precisión en cálculos decimales mediante la biblioteca decimal, lo que permite un control más preciso de los números con decimales y evitar errores de redondeo en cálculos financieros o científicos.

Ejemplo Práctico:

```python
import math

num1 = 10
num2 = 3.14
num3 = 2 + 3j

print(type(num1), type(num2), type(num3))
print(math.sqrt(16)) #Raíz cuadrada
print(math.pi) #Valor de pi
```

2. Cadenas de Caracteres (str)

Las cadenas (`str`) representan texto y se definen con comillas simples o dobles.

Son inmutables: no se modifican en el lugar, sino que cada operación genera una nueva cadena.

Python ofrece operaciones comunes como concatenación, búsqueda y reemplazo, además de métodos como `strip()`, `replace()`, `find()` y `split()`.

También puedes usar secuencias de escape (`\n`, `\t`), cadenas multilínea con comillas triples y formatos modernos con f-strings para construir mensajes de forma legible.

Ejemplo Práctico:

```python
cadena = "Hola, Python"
print(cadena.lower())
print(cadena.upper())
print(cadena.split(,))
nombre = "Carlos"
edad = 30
print(f'Mi nombre es {nombre} y tengo {edad} años.')
```

## 3. Listas (list)

Las listas son colecciones ordenadas y mutables. Se crean con `[]` y aceptan elementos de distintos tipos.

Puedes acceder por índice (desde 0), modificar elementos y aplicar métodos como `append()`, `remove()`, `pop()`, `sort()` o `reverse()`.

Cuando necesitas construir listas de forma compacta y eficiente, la comprensión de listas (`list comprehension`) es una opción muy útil.

Ejemplo práctico:

```python
mi_lista = [1, 2, 3, "cuatro", True]
mi_lista.append(5) # Agregar 5 al final de la lista
mi_lista.remove(2) # Eliminar un elemento específico
print(mi_lista) # Imprimir lista modificada
print(mi_lista[1]) # Acceder al segundo elemento
```

## 4. Diccionarios (dict)

Los diccionarios guardan información como pares clave-valor y se crean con `{}`.

Son muy útiles para búsquedas rápidas por clave y para representar datos estructurados. Las claves deben ser inmutables (por ejemplo, `str`, `int` o tuplas).

Métodos como `keys()`, `values()`, `items()`, `get()` y `update()` facilitan su uso diario.

Ejemplo práctico:

```python
mi_diccionario = {"nombnre" : "Python", "version" : 3.10}
mi_diccionario["año"] = 2024 # Agregar un nuevo par clave-valor
print(mi_diccionario["nombre"]) # Imprimir el valor de una clave
```

## 5. Tuplas (tuple)

Las tuplas son similares a las listas, pero inmutables. Se crean con `()` y sirven para datos que no deberían cambiar.

Permiten acceso por índice y son útiles para representar registros fijos, coordenadas o retornos múltiples de funciones.

Sus métodos principales son `count()` e `index()`.

Ejemplo práctico:

```python
mi_tupla = (10, 20, 30)
print(mi_tupla[1]) # Imprimir el segundo elemento
print(len(mi_tupla)) # Imprimir la longitud de la tupla
```

## 6. Conjuntos (set)

Los conjuntos (`set`) almacenan elementos únicos y no garantizan orden.

Son útiles para eliminar duplicados y para operaciones de teoría de conjuntos como unión, intersección y diferencia.

Métodos frecuentes: `add()`, `remove()`, `union()`, `intersection()` y `difference()`.

Ejemplo práctico:

```python
mi_conjunto = {1, 2, 3, 3, 4, 4, 5}
mi conjunto.add(6) # Agrega un nuevo elemento
mi conjunto.remove(3) # Elimina un elemento
print(mi_conjunto) # Imprime el conjunto sin duplicados
```

## 7. Booleanos (bool)

Los booleanos representan dos valores: `True` y `False`.

Se usan en condicionales y bucles para decidir qué camino sigue el programa. Python también evalúa ciertos valores como falsos (por ejemplo, `0`, `None`, `""`, `[]`), mientras que el resto suele evaluarse como verdadero.

Para combinar condiciones se usan `and`, `or` y `not`.

Ejemplo práctico:

```python
es_mayor = 10 > 5 # True
es_igual = 10 == 5 # False
print(es_mayor and es_igual) # Evaluación lógica
```

## 4. Estructuras de control

### 1. Sentencia if

La sentencia `if` ejecuta un bloque solo cuando una condición es verdadera.

Su estructura básica es:

```python
if condicion:
    # Bloque de código que se ejecuta si la condición es verdadera
```

Para evaluar más casos, puedes usar `elif` y `else`:
Ejemplo práctico:

```python
if condicion1:
    # Código si la condicion1 es verdadera
elif condicion2:
    # Código si la condicio1 es falsa y la condicion2 es verdadera
else:
    # Código si ninguna de las condiciones anteriores se cumple
```

### 2. Sentencia for

El bucle `for` recorre elementos de una secuencia (listas, cadenas, rangos, etc.).

Su estructura es:

```python
for variable in secuencia:
    # Bloque de codigo a ejecutar en cada iteración
```

### 3. Sentencia while

El bucle `while` repite un bloque mientras la condición sea verdadera.

Se usa cuando no sabes de antemano cuántas iteraciones habrá.

Su estructura es:

```python
while condicion:
    # Bloque de código a ejecutar mientras la condicion sea verdadera
```

```python
contador = 0

while contador < 5:
    print(f'Contador: {contador}')
    contador += 1
```

### 4. Sentencias break y continue

`break` y `continue` modifican el comportamiento de los bucles.
- `break`: termina el bucle de inmediato.
- `continue`: salta al siguiente ciclo sin ejecutar el resto del bloque actual.

Ejemplo práctico:
```python
for i in range(10):
    if i == 5:
        break # Se detiene en la ejecución del bucle cuando i es igual a 5
    print(i)

for i in range(10):
    if i % 2 == 0:
        continue # Se salta los números pares
    print(i)
```

### 5. Sentencia pass

`pass` es un marcador de posición. Se usa cuando la sintaxis exige un bloque, pero todavía no quieres implementar lógica.

Ejemplo práctico:

```python
for i in range(5):
    if i == 3:
        pass # No hace nada, pero evita errores de sintaxis
    else:
        print(i)
```

```python
persona1.edad = 31 # Cambiamos el valor de un atributo
```

```python
persona1.apellido = 'Gómez'

```

# 3. Sentencias condicionales

## Introducción a condicionales

Las condicionales permiten que un programa tome decisiones según una condición. Dependiendo del resultado (`True` o `False`), se ejecuta un bloque de código u otro.

### ¿Qué son las condiciones?

Una condición es una expresión booleana. Su resultado define el camino de ejecución del programa. En Python, los bloques básicos para esto son `if`, `elif` y `else`.

### ¿Cómo funcionan las condicionales?

Primero se evalúa la condición del `if`. Si es verdadera, ese bloque se ejecuta. Si no, Python prueba los `elif` y, si ninguno cumple, ejecuta `else` (si existe).

## Tipos de condicionales

- if: Ejecuta un bloque de código si una condición específica es VERDADERO.

Ejemplo en pseudocódigo:

```
SI (edad >= 18) ENTONCES
    ESCRIBIR "Eres mayor de edad"
FIN SI
```

Descripción: El mensaje se imprime solo si `edad >= 18` es verdadero.

Ejemplo en Python:

```python
edad = 18 # Puedes cambiar este valor para probar
if edad >= 18:
    print("Eres mayor de edad")
```

- else: Ejecuta un bloque de código si la condición en el if es FALSO.

Ejemplo en pseudocódigo:

```
SI (edad >= 18) ENTONCES
    ESCRIBIR "Eres mayor de edad"
SINO
    ESCRIBIR "Eres menor de edad"
FIN SI
```

Descripción: Si `edad >= 18` es falso, el programa imprime "Eres menor de edad".

Ejemplo en Python:

```python
edad = 18 # Puedes cambiar este valor para probar
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")
```

- else if: Permite evaluar múltiples condiciones en orden. Si el `if` falla, se prueba la siguiente condición.

Ejemplo en pseudocódigo:

```
SI (nota >= 90) ENTONCES
    ESCRIBIR "Aprobado con distinción"
SINO SI (nota >= 60) ENTONCES
    ESCRIBIR "Aprobado"
SINO
    ESCRIBIR "Reprobado"
FIN SI
```

Descripción: El mensaje cambia según el valor de `nota`.

Ejemplo en Python:

```python
nota = 85 # Puedes cambiar este valor para probar
if nota >= 90:
print("Aprobado con distinción")
elif nota >= 60:
print("Aprobado")
else:
print("Reprobado")
```

### Importancia de las Condicionales

Las condicionales son esenciales porque permiten:
- Tomar decisiones dinámicas según los datos.
- Controlar el flujo de ejecución de forma clara.
- Cubrir distintos escenarios sin duplicar lógica.

## Condicionales simples

Un condicional simple ejecuta un bloque solo cuando se cumple una condición.

### ¿Qué es un Condicional Simple?

Es la forma más básica de decisión en un programa: una condición y un solo bloque de acción cuando esa condición es verdadera.

#### Ejemplos de Condicionales Simples en Pseudocódigo

Ejemplo de condicional simple con mensaje de bienvenida

```
DEFINIR nombre COMO STRING ← "Ana"
SI (nombre = "Ana") ENTONCES
    ESCRIBIR "Hola, Ana! Bienvenida."
FIN SI
```

Descripción: si `nombre` es "Ana", se ejecuta el mensaje; si no, el bloque se omite.

Ejemplo en Python:

```python
nombre = "Ana" # Puedes cambiar este valor para probar
if nombre == "Ana":
    print("Hola, Ana! Bienvenida.")
```

Ejemplo de Condicional Simple con Verificación de Edad

```
DEFINIR edad COMO ENTERO ← 16
SI (edad >= 18) ENTONCES
    ESCRIBIR "Acceso permitido"
FIN SI
```

Descripción: se imprime "Acceso permitido" solo cuando `edad >= 18`.

Ejemplo en Python:

```python
edad = 16 # Puedes cambiar este valor para probar
if edad >= 18:
    print("Acceso permitido")
```

Ejemplo de Condicional Simple con Verificación de Temperatura

```
DEFINIR temperatura COMO ENTERO ← 30
SI (temperatura > 25) ENTONCES
    ESCRIBIR "Hace calor"
FIN SI
```

Descripción: se muestra "Hace calor" cuando la temperatura supera 25.

Ejemplo en Python: 

```python
temperatura = 30 # Puedes cambiar este valor para probar
if temperatura > 25:
    print("Hace calor")
```

### Importancia de los Condicionales Simples

- Facilitan la toma de decisiones: Permiten que el programa tome decisiones basadas en condiciones específicas, lo que ayuda a controlar el flujo de ejecución.
- Simplifican la lógica del programa: Al proporcionar una forma clara y directa de manejar decisiones simples, los condicionales simples ayudan a mantener el código limpio y comprensible.
- Son la base para estructuras más complejas: Son la base sobre la cual se construyen estructuras condicionales más complejas como else if, switch, y combinaciones de múltiples condiciones.

## Condicionales múltiples

Los condicionales múltiples permiten evaluar varias condiciones en orden y ejecutar la primera que se cumple.

### ¿Qué son los condicionales múltiples?

Se implementan con `if`, `elif` y `else`. Son útiles cuando hay más de dos posibles salidas.

```
    ESCRIBIR "Calificación D"

```

```python
calificacion = 75 # Puedes cambiar este valor para probar

else:
```

### Importancia de los Condicionales Múltiples

## Switch

`switch` (en Python, `match-case`) permite comparar una expresión con varios casos de forma ordenada y legible.

### ¿Qué es un switch?

Es una estructura para elegir entre múltiples opciones según un valor. En Python está disponible como `match-case` (Python 3.10+).

#### Ejemplos de switch en Pseudocódigo

Ejemplo de switch con Día de la Semana

```
DEFINIR dia COMO ENTERO ← 3

        ESCRIBIR "Lunes"
    CASO 2:
        ESCRIBIR "Martes"
    CASO 3:
        ESCRIBIR "Miércoles"
    CASO 4:
        ESCRIBIR "Jueves"
    CASO 5:
        ESCRIBIR "Viernes"
    CASO 6:
        ESCRIBIR "Sábado"
    CASO 7:
        ESCRIBIR "Domingo"
    POR DEFECTO:
        ESCRIBIR "Día inválido"
FIN SEGÚN
```

Descripción: compara `dia` con varios casos y usa un caso por defecto para valores no válidos.

Ejemplo en Python:

```python
dia = 3 # Puedes cambiar este valor para probar
match dia:
    case 1:
        print("Lunes")
    case 2:
        print("Martes")
    case 3:
        print("Miércoles")
    case 4:
        print("Jueves")
    case 5:
        print("Viernes")
    case 6:
        print("Sábado")
    case 7:
        print("Domingo")

        print("Día inválido")
```

Descripción: evalúa una opción de menú y ejecuta el caso correspondiente.

```python
    case 1:

        print("Opción 2 seleccionada")
```


### Importancia del switch

- Mejora legibilidad cuando hay muchas ramas por valor.
- Ordena decisiones en torno a una sola expresión.

## Uso de Convenciones de Nombres en Variables (snake_case)

`snake_case` consiste en escribir nombres en minúsculas separados por guion bajo. En Python es la convención recomendada para variables y funciones.

Por ejemplo, en Python, una variable que almacena la edad de un usuario debería declararse de la siguiente manera:

```python
edad_usuario = 25
nombre_completo = "Carlos Pérez"
```

Su uso mejora la consistencia y hace el código más fácil de leer.

### Beneficios del Uso de snake_case

El uso de snake_case en variables tiene múltiples ventajas, entre ellas:
1. Legibilidad mejorada: Facilita la comprensión del código, especialmente en proyectos colaborativos.
2. Estándar recomendado: En lenguajes como Python, la convención oficial PEP 8 sugiere el uso de snake_case en nombres de variables y funciones.
3. Evita errores de interpretación: En comparación con camelCase o PascalCase, el uso de guiones bajos mejora la claridad de los nombres largos.

Por ejemplo, para guardar un correo:

```python
correo_electronico = "usuario@email.com"
```

Si se usa camelCase, se rompe el estilo recomendado en Python:

```python
correoElectronico = "usuario@email.com" # No recomendado en Python
```

### Aplicación Práctica en Código

En un sistema de gestión de empleados, `snake_case` mantiene claridad en nombres:

```python
nombre_empleado = "Ana Gómez"
salario_mensual = 3500.50
fecha_ingreso = "2024-02-16"
```

Los nombres describen bien el propósito y reducen ambigüedad.

### Diferencias Entre Convenciones de Nombres

Convenciones comunes:
- snake_case → Usado en Python y bases de datos (nombre_variable).
- camelCase → Usado en JavaScript y Java (nombreVariable).
- PascalCase → Usado en nombres de clases (NombreVariable).
- kebab-case → Usado en nombres de archivos y URLs (nombre-variable).

Cada una tiene su contexto, pero en variables de Python la opción estándar es `snake_case`.

### Conclusión

Aplicar `snake_case` desde el inicio mejora legibilidad, coherencia y colaboración en equipo.

# 4. Control de flujo en python

## Introducción a los ciclos

En programación, un ciclo (también conocido como bucle) es una estructura que permite repetir un bloque de código varias veces. Los ciclos son fundamentales porque nos permiten automatizar tareas repetitivas, haciendo que el código sea más eficiente y fácil de mantener.

## ¿Por qué son importantes los ciclos? 🤔

Imagina que necesitas imprimir los números del 1 al 100. Sin ciclos, tendrías que escribir una línea de código para cada número, lo cual sería tedioso y propenso a errores. Con un ciclo, puedes lograr esto con solo unas pocas líneas de código.

## Tipos de ciclos
Existen varios tipos de ciclos, pero los más comunes son:

### Ciclo for

Se utiliza cuando sabes cuántas veces quieres repetir una acción. Por ejemplo, para imprimir los números del 1 al 10:

Pseudocódigo
```
PARA i DESDE 1 HASTA 10 HACER
    IMPRIMIR(i)
FIN PARA
```

Python
```python
for i in range(1, 11):
    print(i)
```

### Ciclo while

Se utiliza cuando no sabes exactamente cuántas veces se repetirá la acción, pero tienes una condición que debe cumplirse para continuar. Por ejemplo, para imprimir números hasta que una variable alcance un valor específico:

Pseudocódigo
```
i = 1
MIENTRAS i <= 10 HACER
    IMPRIMIR(i)
    i = i + 1
FIN MIENTRAS
```

Python
```python
i = 1
while i <= 10:
    print(i)
    i = i + 1
```

### Ciclo do-while

Es similar a `while`, pero evalúa la condición al final, por lo que el bloque se ejecuta al menos una vez. En Python no existe de forma nativa y suele simularse con `while True` + `break`.

Ejemplo de suma del 1 al 5:

Pseudocódigo
```
suma = 0
PARA i DESDE 1 HASTA 5 HACER
    suma = suma + i
FIN PARA
IMPRIMIR("La suma es:", suma)
```

Python
```python
suma = 0
i = 1
while True:
    suma += i
    i += 1
    if i > 5:
        break
print("La suma es:", suma)
```

Aquí se acumula la suma en cada iteración y el bucle finaliza al superar el límite.

### Ciclo for

El ciclo `for` repite un bloque sobre una secuencia o rango. Es ideal cuando conoces el recorrido que quieres hacer.

```
    // Bloque de código a repetir
```

Python
```python
for variable in range(valor_inicial, valor_final + 1):
    # Bloque de código a repetir
```

# 5. Estructuras de datos en python

## Estructuras de Datos en Python

### 1. Listas (list)

Las listas son colecciones ordenadas y mutables. Se usan cuando necesitas agregar, quitar o actualizar elementos con frecuencia.

#### 1.1. Creación de listas

Puedes crearlas con `[]` o `list()`. Admiten distintos tipos de datos, incluso listas anidadas.

Crear listas en Python:
```python
lista_vacia = []
lista_numeros = [1, 2, 3, 4, 5]
lista_mixta = [1, "Hola", 3.14, True]
lista_anidada = [[1, 2], [3, 4]]
```

#### 1.2. Acceder a elementos de una lista

El acceso se hace por índice: el primero es `0` y el último es `-1`.

```python
lista = ["Python", "Java", "C++", "JavaScript"]
print(lista[0]) # Python
print(lista[-1]) # JavaScript
```

#### 1.3. Modificar elementos de una lista

Los elementos se modifican asignando un nuevo valor en la posición deseada.

```python
lista = [10, 20, 30]
lista[1] = 50 # Cambia el segundo elemento
print(lista) # [10, 50, 30]
```

#### 1.4. Métodos útiles de listas

Python proporciona varios métodos para manipular listas:
```python
numeros = [1, 2, 3]
numeros.append(4) # Agrega un elemento al final
numeros.insert(1, 10) # Inserta 10 en la posición 1
numeros.remove(2) # Elimina el primer 2 encontrado
ultimo = numeros.pop() # Elimina y retorna el último elemento
print(numeros) # [1, 10, 3]
```

#### 1.5. Iterar sobre una lista

Puedes recorrer una lista con `for` o `while`.
```python
frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
    print(fruta)
```

#### 1.6. Comprensión de listas

La comprensión de listas permite crear resultados de forma compacta y legible.

### 2. Tuplas (tuple)

Las tuplas son similares a las listas, pero inmutables. Son útiles para datos que no deben cambiar.

#### 2.1. Creación de tuplas

Se crean con `()` o con `tuple()`.

Crear tuplas en Python:
```python
tupla_vacia = ()
tupla_numeros = (1, 2, 3, 4, 5)
tupla_mixta = (1, "Hola", 3.14, True)
tupla_anidada = ((1, 2), (3, 4))
```

#### 2.2. Acceder a elementos de una tupla

Se accede a sus elementos por índice, igual que en las listas.
```python
tupla = ("Python", "Java","C++", "JavaScript")
print(tupla[0]) # Python
print(tupla[-1]) # JavaScript
```

#### 2.3. Métodos útiles de tuplas

Las tuplas tienen menos métodos que las listas, ya que son inmutables. Sin embargo, los más utilizados son:
```python
tupla = (1, 2, 3, 1, 4, 1)
print(tupla.count(1)) # Cuenta cuántas veces aparece el número 1
print(tupla.index(3)) # Devuelve el índice donde aparece el número 3
```

#### 2.4. Iterar sobre una tupla

Se puede recorrer con un bucle `for`.
```python
for elemento in tupla:
    print(elemento)
```

### 3. Conjuntos (set)

Los conjuntos almacenan elementos únicos sin orden fijo. Son ideales para eliminar duplicados y hacer operaciones de pertenencia e intersección.

#### 3.1. Creación de conjuntos

Se crean con `set()` o con llaves `{}` cuando ya hay elementos.

Crear conjuntos en Python:
```python
conjunto_vacio = set()
conjunto_numeros = {1, 2, 3, 4, 5}
conjunto_mixto = {1, "Hola", 3.14, True}
```

#### 3.2. Acceder a elementos de un conjunto

Como no tienen orden, no se accede por índice. Para recorrerlos, usa un `for`.
```python
conjunto = {"Python","Java","C++","JavaScript"}
for elemento in conjunto:
    print(elemento)
```

#### 3.3. Métodos útiles de conjuntos

Python ofrece varios métodos para manipular conjuntos:
```python
numeros = {1, 2, 3}
numeros.add(4) # Agrega un elemento al conjunto
numeros.remove(2) # Elimina el elemento 2 (da error si no existe)
numeros.discard(5) # Intenta eliminar 5 (no da error si no existe)
numeros.clear() # Elimina todos los elementos
```


## Comparación y selección de estructuras de datos

### Comparación de listas, tuplas, conjuntos y diccionarios

| Estructura | Mutable | Indexada | Permite Duplicados | Claves Únicas | Rendimiento |
| --- | --- | --- | --- | --- | --- |

- Diccionarios: Cuando se necesita almacenar datos asociados a claves únicas y acceder rápidamente a los valores.

## Introducción funciones

## Paso de parámetros

El paso de parámetros define cómo una función recibe datos para trabajar. En Python, esto depende del tipo de objeto y de si se modifica o no dentro de la función.

Como referencia conceptual, se suele explicar así:
1. Paso por valor: En este método, se pasa una copia del valor del argumento a la función. Esto significa que cualquier modificación realizada al parámetro dentro de la función no afecta el valor original fuera de la función. Es decir, los cambios se limitan al ámbito local de la función.
2. Paso por referencia: En este caso, se pasa una referencia al argumento en lugar de su valor. Esto significa que cualquier cambio realizado al parámetro dentro de la función afectará el valor original fuera de la función. El paso por referencia es útil cuando se necesita que la función modifique directamente el valor original del argumento.

### Ejemplos de paso de parámetros en pseudocódigo y python

Ejemplo 1: Paso por valor

En este ejemplo, una función que suma un número dado se demostrará con paso por valor.

Pseudocódigo:
```
PROCEDIMIENTO incrementar_por_valor(numero)
    numero ← numero + 1
    ESCRIBIR "Valor dentro del procedimiento: " + numero
FIN PROCEDIMIENTO
DEFINIR x COMO ENTERO ← 5
incrementar_por_valor(x)
ESCRIBIR "Valor fuera del procedimiento: " + x
```
Salida:
```
Valor dentro del procedimiento: 6
Valor fuera del procedimiento: 5
```

Python:
```python
def incrementar_por_valor(numero):
    numero += 1
    print("Valor dentro del procedimiento:" , numero)
x = 5
incrementar_por_valor(x)
print("Valor fuera del procedimiento:" , x)
```

Salida:
```
Valor dentro del procedimiento: 6
Valor fuera del procedimiento: 5
```

Aunque `numero` cambia dentro de la función, `x` fuera de ella se mantiene igual.

Ejemplo 2: Paso por referencia

Aquí una lista se modifica dentro del procedimiento.

Pseudocódigo:
```
PROCEDIMIENTO agregar_elemento_por_referencia(lista)
    AGREGAR 4 A lista
    ESCRIBIR "Lista dentro del procedimiento: " + lista
FIN PROCEDIMIENTO
DEFINIR numeros COMO LISTA ← [1, 2, 3]
agregar_elemento_por_referencia(numeros)
ESCRIBIR "Lista fuera del procedimiento: " + numeros
```
Salida:
```

Lista fuera del procedimiento: [1, 2, 3, 4]

```

La modificación sobre la lista queda visible fuera de la función porque es un objeto mutable.

## Introducción procedimientos

Un procedimiento ejecuta acciones sin enfocarse en devolver un resultado. En Python, se escribe igual que una función (`def`), pero normalmente se usa para tareas con efectos directos, como imprimir o actualizar estado.

Ayuda a separar responsabilidades y reducir repetición.

### ¿Cómo se arman los procedimientos?

Estructura básica:
1. Definición con `def`.
2. Parámetros opcionales.
3. Cuerpo con acciones a ejecutar.

### Ejemplos de procedimientos en pseudocódigo y python

## Ámbitos de las variables

El ámbito (scope) define dónde una variable existe y puede utilizarse.

En Python conviene entender la regla LEGB: Local, Enclosing, Global y Built-in. Python busca nombres en ese orden.

Dos ámbitos clave son global y local:
1. Ámbito global: Una variable con ámbito global es accesible desde cualquier parte del programa. Estas variables se definen fuera de cualquier función o bloque de código. Son útiles cuando se necesita mantener un valor o estado accesible en múltiples partes de un programa. Sin embargo, el uso excesivo de variables globales puede llevar a un código difícil de mantener y a errores impredecibles.
2. Ámbito local: Una variable con ámbito local sólo es accesible dentro del bloque de código o la función en la que se define. Esto significa que una vez que la ejecución del programa sale del bloque donde se declaró la variable, la variable deja de existir. Las variables locales ayudan a evitar conflictos de nombres y mantienen el código modular y fácil de entender.

### Ejemplos de ámbitos de variables en pseudocódigo y python

Ejemplo 1: Variable de ámbito global

Pseudocódigo:

```
    ESCRIBIR "El valor de x es: " + x
FIN PROCEDIMIENTO
mostrar_valor_global() // Salida: El valor de x es: 10
```
Python:
```python
x = 10 # Variable global

```

`x` es global porque está fuera de la función.

Ejemplo 2: Variable de ámbito local

Pseudocódigo:
```
PROCEDIMIENTO calcular_suma()
    DEFINIR a COMO ENTERO ← 5 // Variable local
    DEFINIR b COMO ENTERO ← 3 // Variable local
    ESCRIBIR "La suma es: " + (a + b)
FIN PROCEDIMIENTO
calcular_suma() // Salida: La suma es: 8
ESCRIBIR a // Error: 'a' no está definido
```
Python:
```python

b = 3 # Variable local

```

Ejemplo 3: Usando variables locales y globales juntas

Dominar el ámbito evita conflictos de nombres y errores difíciles de depurar.

## Recursividad

La recursividad consiste en que una función se llame a sí misma para resolver subproblemas del problema original.

Toda función recursiva debe tener dos partes:
1. Caso base: Esta es la condición de terminación que detiene las llamadas recursivas. Sin un caso base, la función se llamaría indefinidamente, llevando a un desbordamiento de la pila (stack overflow).
2. Caso recursivo: reduce el problema en cada llamada hasta llegar al caso base.

### Ejemplos de Recursividad en Pseudocódigo y Python

Ejemplo 1: Factorial de un número

El factorial de un número entero positivo n se define como el producto de todos los números enteros positivos desde 1 hasta n. El factorial de n (denotado como n!) se puede calcular recursivamente como:
- n! = n * (n-1)! para n > 0
- 0! = 1 (caso base)

Pseudocódigo:
```
FUNCION factorial(n)
    SI n = 0 ENTONCES
        RETORNAR 1 // Caso base
    FIN SI
    RETORNAR n * factorial(n - 1) // Caso recursivo
FIN FUNCION
ESCRIBIR "El factorial de 5 es: " + factorial(5)
```
Salida:
```
El factorial de 5 es: 120
```
Python:
```python
def factorial(n):
    if n == 0:
        return 1 # Caso base
    return n * factorial(n - 1) # Caso recursivo
print("El factorial de 5 es:", factorial(5))
```
Salida:
```
El factorial de 5 es: 120
```
En este ejemplo, la función factorial se llama a sí misma con un valor decreciente de n hasta que n es 0, momento en el cual retorna 1 como resultado.

Ejemplo 2: Secuencia de fibonacci

La secuencia de Fibonacci es una sucesión de números donde cada número es la suma de los dos anteriores. Matemáticamente, se define como:
- F(0) = 0, F(1) = 1 (casos base)
- F(n) = F(n-1) + F(n-2) para n > 1 (caso recursivo)

Pseudocódigo:
```
FUNCION fibonacci(n)
    SI n = 0 ENTONCES
        RETORNAR 0 // Caso base
    FIN SI
    SI n = 1 ENTONCES
        RETORNAR 1 // Caso base
    FIN SI
    RETORNAR fibonacci(n - 1) + fibonacci(n - 2) // Caso recursivo
FIN FUNCION
ESCRIBIR "El término 6 de Fibonacci es: " + fibonacci(6)
```
Salida:
```
El término 6 de Fibonacci es: 8
```
Python:
```python
def fibonacci(n):
    if n == 0:
        return 0 # Caso base
    if n == 1:
        return 1 # Caso base
    return fibonacci(n - 1) + fibonacci(n - 2) # Caso recursivo
print("El término 6 de Fibonacci es:", fibonacci(6))
```
Salida:
```
El término 6 de Fibonacci es: 8
```
En este ejemplo, la función fibonacci se llama a sí misma dos veces en cada llamada, calculando fibonacci(n-1) y fibonacci(n-2) hasta que alcanza los casos base 0 y 1.

Ejemplo 3: Búsqueda binaria

La búsqueda binaria es un algoritmo eficiente para encontrar un elemento en una lista ordenada. Funciona dividiendo repetidamente el rango de búsqueda a la mitad hasta que encuentra el elemento o determina que no está en la lista. La búsqueda binaria se puede implementar de forma recursiva.

Pseudocódigo:
```
FUNCION busqueda_binaria(lista, inicio, fin, objetivo)
    SI inicio > fin ENTONCES
        RETORNAR -1 // Caso base: No encontrado
    FIN SI
    DEFINIR medio COMO ENTERO ← (inicio + fin) / 2
    SI lista[medio] = objetivo ENTONCES
        RETORNAR medio // Caso base: Encontrado
    SINO SI lista[medio] > objetivo ENTONCES
        RETORNAR busqueda_binaria(lista, inicio, medio - 1, objetivo) // Caso recursivo
    SINO
        RETORNAR busqueda_binaria(lista, medio + 1, fin, objetivo) // Caso recursivo
    FIN SI
FIN FUNCION
DEFINIR lista COMO LISTA ← [1, 3, 5, 7, 9, 11]
ESCRIBIR "El índice de 7 es: " + busqueda_binaria(lista, 0, LONGITUD(lista) - 1, 7)
```
Salida:
```
El índice de 7 es: 3
```
Python:
```python
def busqueda_binaria(lista, inicio, fin, objetivo):
    if inicio > fin:
        return -1 # Caso base: No encontrado
    medio = (inicio + fin) // 2
    if lista[medio] == objetivo:
        return medio # Caso base: Encontrado
    elif lista[medio] > objetivo:
        return busqueda_binaria(lista, inicio, medio - 1, objetivo) # Caso recursivo
    else:
        return busqueda_binaria(lista, medio + 1, fin, objetivo) # Caso recursivo
lista = [1, 3, 5, 7, 9, 11]
print("El índice de 7 es:", busqueda_binaria(lista, 0, len(lista) - 1, 7))
```
Salida:
```
El índice de 7 es: 3
```
En este ejemplo, la función busqueda_binaria utiliza la recursividad para reducir el rango de búsqueda a la mitad hasta encontrar el objetivo o determinar que no existe en la lista.

La recursividad es una poderosa herramienta en la programación que permite resolver problemas de forma elegante y concisa. Sin embargo, también puede ser más compleja de entender y manejar, especialmente cuando no se establece correctamente un caso base, lo que puede llevar a un desbordamiento de la pila. Entender cómo y cuándo usar la recursividad es clave para escribir algoritmos eficientes y efectivos.
