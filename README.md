# 🚀 Desarrollo Web Fullstack Django Python Trainee

Bienvenido a este repositorio de apuntes sobre desarrollo web 📚

![Django](https://external-content.duckduckgo.com/iu/?u=https%3A%2F%2Fdatascientest.com%2Fwp-content%2Fuploads%2F2023%2F05%2Fdjango-datascientest.png&f=1&nofb=1&ipt=6e95ec82169abb328f062e9f1e237afcffa299f1bc6546cac9881afab5bffcd0)

Aquí encontrarás contenidos relacionados al desarrollo de aplicaciones web utilizando el framework Django. El material esta disponible en archivos *markdown* organizados para facilitar el aprendizaje progresivo.

---

## 🎯 Objetivo del curso

Entender las bases del desarrollo web fullstack, comprendiendo conceptos, lenguajes y frameworks utilizados en el desarrollo **frontend** y **backend** de las aplicaciones web.

---

## 🧠 Metodología

El contenido está organizado de manera **incremental**, comenzando desde los fundamentos hasta temas más avanzados, permitiendo una curva de aprendizaje sólida y progresiva.

---

## 📂 Contenidos del curso

### 1. 🌐 Fundamentos de Desarrollo Frontend
Introducción a las tecnologías base del desarrollo web:

- HTML5 🏗️
- CSS3 🎨
- JavaScript ⚡
- Bootstrap 📱
- jQuery 🔧

---

### 2. 🐍 Fundamentos de Programación en Python
Conceptos esenciales para comenzar a programar:

- Variables y tipos de datos
- Estructuras de control
- Funciones

---

### 3. ⚙️ Programación Avanzada en Python
Profundización en el lenguaje:

- Programación orientada a objetos (POO)
- Manejo de errores
- Módulos y paquetes
- Manejo de archivos
- Buenas prácticas y patrones

---

### 4. 🗄️ Fundamentos de Bases de Datos Relacionales
Bases teóricas y prácticas de almacenamiento de datos:

- Modelado de datos
- SQL básico e intermedio
- Relaciones (1-1, 1-N, N-N)
- Normalización

---

### 5. 🌍 Desarrollo de Aplicaciones Web con Django
Construcción de aplicaciones backend con Django:

- Arquitectura MVT
- Creación de proyectos y apps
- Vistas y templates
- Formularios
- Autenticación de usuarios

---

### 6. 🔗 Acceso a Datos en Aplicaciones con Django
Integración y manejo de datos:

- ORM de Django
- Consultas avanzadas
- Relaciones entre modelos
- Integración con bases de datos

---

## 🛠️ Requisitos

- Conocimientos básicos de computación
- Python 3.x instalado
- Editor de código (VS Code recomendado)
- Git y GitHub
